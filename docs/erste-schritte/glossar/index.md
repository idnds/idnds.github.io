# Glossar

## K

### Kubernetes

Kubernetes ist wie ein automatisierter Regisseur für Ihre Container-Anwendungen. Es kümmert sich darum, dass sie reibungslos laufen, sich bei Bedarf automatisch vermehren und sich selbst reparieren können. So können Entwickler sich auf das Schreiben von Code konzentrieren, während Kubernetes die ganze Arbeit im Hintergrund erledigt, um sicherzustellen, dass die Anwendungen immer verfügbar und skalierbar sind. Kurz gesagt, Kubernetes macht das Verwalten von Anwendungen in der Cloud einfacher und effizienter.

## L

### LTS-Version

LTS steht für "Long-Term Support" (Langzeitunterstützung). Eine LTS-Version ist eine spezielle Version einer Software, die eine längere Supportperiode hat als normale Versionen. Im Allgemeinen erhalten LTS-Versionen regelmäßige Sicherheitsupdates, Fehlerbehebungen und manchmal auch neue Funktionen über einen längeren Zeitraum hinweg, typischerweise mehrere Jahre.

LTS-Versionen sind besonders wichtig für Unternehmen und Organisationen, die Stabilität und Kontinuität in ihren Systemen benötigen. Durch die Verwendung einer LTS-Version können sie sicherstellen, dass ihre Systeme über einen längeren Zeitraum unterstützt werden, was Zeit und Ressourcen spart.

Für Endbenutzer bedeutet eine LTS-Version in der Regel eine zuverlässigere und konsistentere Softwareerfahrung über einen längeren Zeitraum hinweg, da die Entwickler sich darauf konzentrieren, die Version stabil zu halten und kritische Probleme zu beheben.
